## 소개

이 프로젝트는 스프링부트를 이용한 RESTful API 서버와 Vue.js를 이용한 프론트엔드 애플리케이션으로 구성되어 있습니다.

## 기술 스택

- **백엔드**: Spring Boot, MySQL
- **프론트엔드**: Vue.js, Vue Router, Axios
- **빌드 도구**: Maven, Node.js, npm

## 설치

### 사전 요구 사항

- Java 11 이상
- Maven
- Node.js (v14 이상)
- npm
